## 概要
**概要**  
AI を化学分野に導入するには大量の教師データが必要です。既存の分子データだけでなく、将来存在しうる未知の分子についても学習させる必要があります。本研究では、計算機上での高分子生成・合成・物性評価を視野に入れ、糖類に着目します。未知糖の存在可能性や物性を評価するために、**AmberTools** と **LAMMPS** を用いて分子構造の検証とシミュレーションを行います。

**目的と意義**  
未知の糖分子を含む広範な化学空間をモデル化し、AI による材料設計や性質予測のための高品質なデータセットを構築することを目指します。これにより、既存データに依存しない汎用的な学習基盤を提供し、将来的な分子設計の探索範囲を拡張します。

## 必要条件
- **OS**: Ubuntu 20.04 / Windows 10 (WSL 推奨)  
- **ソフトウェア**: LAMMPS, AmberTools, Python 3.10+, git, git-lfs  
- **Python パッケージ（例）**: numpy; matplotlib  
- **ハードウェア**: シミュレーション規模に応じたメモリとストレージ

cd GluDimer

# Git LFS を使う場合（大きなファイルがある場合は必須）
git lfs install

```
